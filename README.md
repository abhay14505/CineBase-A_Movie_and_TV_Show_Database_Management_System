CineBase is a database-driven application designed to store, manage, and analyze information related to Movies, TV Shows, Seasons, Episodes, Actors/Directors, Genres, Awards, and User Reviews.  
It is built using MySQL, Python, and Tkinter, implementing core DBMS concepts such as normalization, relational mapping, stored procedures, triggers, and advanced analytical queries.

---

## Features

### Database Features
- Fully normalized relational schema  
- Movie, TV Show, Season, and Episode management  
- Person entity for actors, directors, and creators  
- Genre and Award management  
- User review and rating system  
- Many-to-many relationship handling through associative tables  
- Trigger-based automatic logging  
- Stored procedures and functions for automation  

### GUI Features (Tkinter)
- Role-based access: Admin and User  
- Table viewer with dynamic loading  
- CRUD operations (Admin only)  
- Execution of stored procedures and user-defined functions  
- Built-in advanced SQL queries (join, aggregate, nested)  
- Trigger log viewer  
- Custom SQL query execution
